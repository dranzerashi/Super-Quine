# Super-Quine 

Write A program to print itself n(-10 <= n <= 10) times.

Negative count means the reversed program.

Input

A line contains an integer n.

Output

The required string described above.

Score

Score is the length of your code.

Example

Input:
-2

Source:
abcdefg

Output:
gfedcbagfedcba

Score:
7
